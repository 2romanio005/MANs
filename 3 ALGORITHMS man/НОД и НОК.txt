НОК 10 и 5 = 10 - общий знаминатель
НОД 10 и 5 = 5  - общий делитель
 
НОК - произвередние чесел / их НОД
Числа взаимно обратные или у них нет общего делитель(НОД == 1)

Алгоритм Эвклида:
int NOD(int a, int b){
	while(b != 0){
		swap(a %= b, b);
	}
	return a;
}


int NOK(int a, int b){
	return (a * b) / NOD(a, b);
}
